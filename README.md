# Union Vocera Tracker

A mobile-friendly application developed for Union Hospital to streamline the management, checkout, and monitoring of Vocera badges. This system provides an intuitive interface for both staff members checking out devices and managers monitoring badge inventory and usage.

## Features

- **Staff Checkout System:** Simple, quick interface for staff to check out and return Vocera badges.
- **Manager Dashboard:** Comprehensive view for managers to monitor current badge status, track history, and manage inventory.
- **Mobile-Friendly Design:** Fully responsive interface designed with a professional vanilla CSS design system, ensuring usability across all devices.
- **Real-time Database:** Powered by Supabase for instantaneous updates and secure data management.

## Vibe Coding Stack

This project was developed using modern web technologies and AI-assisted development:

- **Frontend:** Next.js
- **Backend/Database:** Supabase
- **AI Assistant:** Antigravity

## How to Run

To run the development server locally:

1. Install dependencies (e.g., `npm install`)
2. Set up your Supabase environment variables in a `.env.local` file
3. Run the application:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
